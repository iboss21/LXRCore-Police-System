## üì• Installation

### Requirements

- **RedM Server** (Latest version recommended)
- **Framework**: RSG-Core or LXRCore
- **Database**: MySQL/MariaDB
- **Dependencies**:
  - `oxmysql` or `mysql-async`
  - `rsg-target` (optional, for target system)
  - `rsg-inventory` or compatible

### Quick Install

1. **Download** the latest release
   ```bash
   git clone https://github.com/iboss21/LXRCore-Police-System.git tlw-lawman
   ```

2. **Place** in your resources folder
   ```
   resources/[law]/tlw-lawman/
   ```

3. **Import** SQL files
   ```bash
   # Import all migration files in order
   mysql -u username -p database_name < sql/migrations/001_mdt_citizens.sql
   mysql -u username -p database_name < sql/migrations/002_mdt_warrants.sql
   # ... continue for all migration files
   ```

4. **Configure** the system
   - Edit `config/config_main.lua`
   - Edit `config/config_advanced.lua`
   - Set your Discord webhook
   - Customize stations and jobs

5. **Add** to server.cfg
   ```cfg
   ensure rsg-core  # or lxrcore
   ensure oxmysql
   ensure tlw-lawman
   ```

6. **Restart** your server

### Framework Configuration

## üéØ Historical Accuracy

### 1899 Law Enforcement

This system is based on actual Wild West law enforcement:

#### Sheriff
- Elected county official
- Jurisdiction over entire county
- Could deputize civilians
- Responsible for tax collection
- Operated county jail

#### US Marshal
- Federal appointee
- Territory-wide jurisdiction
- Pursued federal criminals
- Served federal warrants
- Protected federal property

#### State Rangers
- State law enforcement
- Frontier protection
- Elite horsemen and trackers
- Military-style organization
- Legendary reputation (Texas Rangers)

#### Town Marshal
- Appointed by town mayor/council
- Town jurisdiction only
- Enforced local ordinances
- Kept the peace
- Less prestigious than Sheriff

### Period-Accurate Crimes

All crimes are based on actual 1899 laws:

- **Horse Theft** - Capital offense (hanging)
- **Cattle Rustling** - Serious felony
- **Train Robbery** - Federal crime
- **Claim Jumping** - Mining law violation
- **Moonshining** - Prohibition violations
- **Vagrancy** - Loitering laws
- **Brawling** - Public order offense

### Communication Methods

All dispatch methods are historically accurate:

- **Telegraph** - Primary long-distance communication
- **Messenger** - Physical message delivery
- **Bell Tower** - Town alarm system
- **Signal Fires** - Long-distance alerts
- **Police Whistle** - Officer signaling
